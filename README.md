# emotion-aware-mental-health-detection
Developed an AI powered multimodal mental health detection system that analyzes emotions from text, speech, facial expressions, and psychological assessments (PHQ-9 and GAD-7) to identify anxiety, stress, and depression.
Leveraged VAE-enhanced feature representation and Reinforcement Learning to improve prediction accuracy and severity estimation, while generating automated reports with risk classification, assessment results, and personalized recommendations.
